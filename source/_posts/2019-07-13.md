## 要点

求对象中多个属性的和
```js
let data = [
  {
    tqaModifyCount: 9,
    tqaMarkCount: 19
  },
  {
    tqaModifyCount: 3,
    tqaMarkCount: 3
  }
]
let tgt = {
  tqaModifyCounts: 12
  tqaMarkCounts: 22
}
```
<!--more-->

## 代码

```js
function sum (dataArr) {
  return dataArr.reduce((acc, obj) => {
    return {
      tqaModifyCounts: acc.tqaModifyCount + obj.tqaModifyCount,
      tqaMarkCounts: acc.tqaMarkCount + obj.tqaMarkCount
    }
  })
}
```
