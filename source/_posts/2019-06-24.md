### 题目

```js
const data = [
  { 'id': 1, name: 'a', 'code': 11 },
  { 'id': 2, name: 'b', 'code': 22 },
  { 'id': 3, name: 'c', 'code': 33 },
]
```
以code值为key的结果对象

<!--more-->

### 分析

主要是`[]`的用法，灵活的很呐！

### 代码

```js
let getKey = (data, key) => {
  return data.map(item => {
    return {
      [item[key]]: item
    }
  })
}
```
