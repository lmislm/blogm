发表于 2018-02-24 | 分类于 学习 | |

题目

给定整数a~1~,a~2~ ,…,a~n~，判断是否可以从中选出若干数，使他们的和恰好为k。

限制条件

  • 1≤ n ≤20
  • -10^8^ ≤ a~i~ ≤ 10^8^
  • -10^8^ ≤ k ≤ 10^8^
    输入
  • n=4
  • a={1,2,4,7}
  • k=13
    输出
    yes （13 = 2 + 4 + 7）
