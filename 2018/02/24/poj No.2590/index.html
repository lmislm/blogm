POj No.2590 Steps

题目

在数轴上的两点之间，按照下面规则移动。

  • 每一步的步幅都要是自然数，同时必须与前一步幅相同或者是差1。

  • 每一步和最后一步的步幅必须是1。
    当给出两点位置的时候，请回答从第一个点移动到第二点需要的最小步数。

    输入和输出

    测试数据个数n、开始地点x以及终点y（0<=x<=y<2^31^),格式如下：

    n
    x~1~ y~1~
    x~2~ y~2~
    …
    x~n~ y~n~

  • 输入

    3
    45 48
    45 49
    45 50

  • 输出

    3
    3
    4
